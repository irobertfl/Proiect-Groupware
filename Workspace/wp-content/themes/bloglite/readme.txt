== Description ==

Bloglite is designed for blogs, magazines, daily newspapers and for writers in general.It is a child theme of bizlte multipurpose wordpress theme. Bloglite is optimized the SEO (search engine optimization) alot so it’s easy to find for websites that use our blogging theme on Google and other search engines. Bloglite is superior to display lot’s ofads, you can have them in the sidebar, footer and more, all through widgets so it’s easy to ingegrate ad programs such as AdSense. Blogllite is compatible with Gutenberg plugin. It’s also compatible with all kind of page builders like King composer, Divi builder, beaver builder, Elementor, visual comper etc. Fully responsive and retina ready design.  Google font support and based bootstrap framework.  Perfect for business, agency, blog, magazine, personal blog, construction, hospital, tarveler, fashion, photography websites.


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
